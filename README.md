## 💻 Design and Analysis of Algorithms (DAA) - C++ Programs

This repository contains my collection of **C++ programming language** implementations for fundamental algorithms and advanced data structures, completed as part of my **Engineering Semester 4** course on **Design and Analysis of Algorithms (DAA)**.

These programs focus on applying various algorithmic techniques—such as **Dynamic Programming**, **Greedy Approach**, **Divide and Conquer**, and **Backtracking**—to solve complex computational problems and analyze their efficiency.

### ✨ Key Focus Areas

  * **Algorithmic Techniques:** Dynamic Programming, Greedy Approach, Divide and Conquer, and Backtracking.
  * **Time Complexity Analysis:** Programs are designed to allow for the verification and analysis of time complexity.
  * **Advanced Data Structures:** Implementation and use of structures like AVL Trees and Graph representations.

### 📂 Repository Contents

The table below provides details on the algorithms and techniques implemented in each C++ source file:

| Filename | Concept / Algorithmic Technique | Detailed Description |
| :--- | :--- | :--- |
| `01knapsack.cpp` | **Dynamic Programming** | Solution for the **0/1 Knapsack Problem** using the Dynamic Programming approach. |
| `avl.cpp` | **AVL Tree (Dictionary)** | Implementation of a self-balancing **AVL Tree** to serve as an efficient Dictionary (key-value store). |
| `graph.cpp` | **Graph Algorithms** | Implementation of a **Facebook social graph** using an **Adjacency List** representation, featuring **Breadth-First Search (BFS)** and **Depth-First Search (DFS)** traversals. |
| `hashing.cpp` | **Hashing (Student Database)** | Implementation of **Hashing** techniques to efficiently store and retrieve student database records. |
| `huffman.cpp` | **Greedy Approach** | Implementation of **Huffman Coding** for data compression using a **Greedy** algorithmic strategy. |
| `mobile.cpp` | **Divide and Conquer / Search / Sort Analysis** | A multi-part program demonstrating and analyzing the complexity of various algorithms on mobile user data (MobileNo, BillAmount):<br>a. **Recursive Quick Sort** (Descending MobileNo)<br>b. **Heap Sort** (Ascending BillAmount)<br>c. **Iterative/Recursive Binary Search**<br>d. Time complexity verification (passes/comparisons) for different test cases. |
| `nqueen.cpp` | **Backtracking** | Solution for the **N-Queens Problem** using the **Backtracking** algorithmic technique. |
| `prims.cpp` | **Greedy Approach** | Implementation of **Prim's Algorithm** for finding the **Minimum Spanning Tree (MST)** of a weighted, undirected graph. |

### 🛠️ Getting Started

These programs are written in standard C++ and should be compatible with any modern C++ compiler (like g++ or Clang).

#### **Prerequisites**

You will need a C++ compiler installed on your system.

  * **Recommended:** [g++ (GNU C++ Compiler)](https://gcc.gnu.org/)

#### **Compiling and Running**

Follow these steps for a typical program, using `01knapsack.cpp` as an example:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/aradhya0507/Design-and-Analysis-of-Algorithm.git
    ```
2.  **Compile the file:** (Use the `-std=c++11` flag or higher for modern C++ features)
    ```bash
    g++ 01knapsack.cpp -o knapsack
    ```
3.  **Run the executable:**
    ```bash
    ./knapsack
    ```
