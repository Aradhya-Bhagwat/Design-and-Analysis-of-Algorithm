Welcome to my repository for **Design and Analysis of Algorithms** implemented in **C++**. This repository contains a collection of fundamental algorithms and dynamic programming solutions developed during my 4th Semester Engineering curriculum.

## 🎯 Objective
The primary goal of this repository is to demonstrate a solid understanding of core computer science concepts by implementing foundational algorithms from scratch in the C++ programming language.

## 🚀 Getting Started

To run these programs locally on your machine, follow the instructions below.

### Prerequisites
You will need a standard C++ compiler installed on your system.
* **Windows:** Install [MinGW](http://www.mingw.org/) or [TDM-GCC](https://jmeubank.github.io/tdm-gcc/).
* **Linux:** GCC/G++ is usually pre-installed. (If not, run `sudo apt install build-essential`).
* **macOS:** Install Xcode command line tools (`xcode-select --install`).

### Compilation & Execution
1. **Clone the repository:**
   ```bash
   git clone https://github.com/Aradhya-Bhagwat/Design-and-Analysis-of-Algorithm.git
   cd Design-and-Analysis-of-Algorithm
   ```
2. **Compile a file** (using `avl.cpp` as an example):
   ```bash
   g++ avl.cpp -o avl
   ```
3. **Run the compiled executable:**
   * On Windows:
     ```bash
     avl.exe
     ```
   * On Linux/macOS:
     ```bash
     ./avl
     ```
